About Me

I'm a software developer from Shepherdstown, WV.

I got into this field because I wanted to make video games. However, as I learned more about the industry, I realized that wasn't for me. I almost lost interest in computer science completely... Until I found a flier for an app development challenge at my university. That was when I made my first iOS application with my best friend. Together we created, Stroboscope, won first place at the app challenge.

From then on, I wanted to be an iOS developer. We made two more iOS applications together, one is an natural language processor themed around Star Trek: TNG and the other is a student resource application for West Virginia University (my Alma Matr). And I've been working on some other projects as well... my best one is PRT Monitor. Hopefully this website will turn into a blog as well as a portfolio, soon.
